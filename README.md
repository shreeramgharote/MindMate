# MindMate

AI-Powered Mental Wellness Assistant for Students

## SDG Goal

SDG 3: Good Health and Well-Being

## Problem Statement

Many students experience stress, anxiety, burnout, and academic pressure but often lack immediate support and guidance.

## Solution

MindMate is an AI-powered chatbot developed using Botpress that provides emotional support, stress-management guidance, study tips, and mental wellness resources.

## Features

- Stress Management Support
- Anxiety Support
- Motivation Assistance
- Study Tips
- Mental Health Resources
- Emergency Response Guidance

## Technology Stack

- Botpress
- HTML
- CSS
- JavaScript
- GitHub

## Future Scope

- Voice-Based Interaction
- Mobile Application
- Sentiment Analysis
- Multilingual Support
- Counselor Integration
